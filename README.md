Esse é um programa de montagem de peças (Softwares de CAD e Engenharia) onde é possível copiar um conjunto já projetado para usar em outro projeto. 
A clonagem profunda permite modificar as dimensões ou o material de um componente específico na nova montagem sem deformar o projeto original da biblioteca.
Foi feito com o padrão de software prototype
